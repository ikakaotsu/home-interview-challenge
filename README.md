# Digiventures Take Home Exercise

## Run Locally

Clone the project

```bash
  git clone https://github.com/ikakaotsu/home-interview-challenge.git
```

Go to the project directory

```bash
  cd home-interview-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## 🙌🏼 ES-es
